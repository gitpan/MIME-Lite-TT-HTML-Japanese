=encoding utf-8

=head1 名前

MIME::Lite::TT::HTML::Japanese - Template-Toolkit によるHTMLメール作成モジュール

=head1 概要

    use MIME::Lite::TT::HTML::Japanese;

    my $msg = MIME::Lite::HTML::Japanese->new(
        From        => 'from@example.com',
        To          => 'to@example.com',
        Subject     => 'Subject',
        Template    => {
            html => 'mail.html',
            text => 'mail.txt',
        },
        TmplOptions => \%options,
        TmplParams  => \%params,
        Icode       => 'euc',    # input characterset (Jcode format)
    );
    
    $msg->send;

=head1 説明

C<MIME::Lite> と C<Template> をつかって HTML メールを作成するモジュール。

=head1 メソッド

=over 4

=item new( %options )

%option を元に MIME::Lite でメールを作成し、その MIME::Lite オブジェクトを返す。

%option のとれる値。

=back
